## 游戏说明
使用金币购买笨璃璃，然后进行一系列调教，得到成熟的笨璃璃（开发出特长），然后卖给领地的建筑，获得金币，从而购买新的笨璃璃。
### 注意事项
- 游玩时请关闭隐私模式，否则可能会出现奇怪的bug。localStorage在浏览器的隐私模式下面是不可读取的。
